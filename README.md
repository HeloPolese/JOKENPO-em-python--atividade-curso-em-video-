# JOKENPO-em-python--atividade-curso-em-video-
Projeto em Python desenvolvido como atividade do Jokenpô (Pedra, Papel e Tesoura) do curso Curso em Vídeo, ministrado pelo professor Guanabara. O programa permite que o usuário jogue contra o computador, utilizando lógica condicional, geração de números aleatórios e interação via terminal.
